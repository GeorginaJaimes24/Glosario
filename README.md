# Glosario delitos cibernéticos

  ## Informática 
  Ciencia de la información y acrónimo de "INFORmación autoMÁTICA". 
  ## Computadora 
  Dispositivo electrónico capaz de recibir datos, procesarlos y entregar resultados en la forma deseada a través de la ejecución de cálculos matemáticos.
## Dato
Carácteres que carecen de sentido y valor semántico.
## Información
Conjunto de datos con un significado de utilidad.
## Informática forense
Ciencia que tiene por objeto adquirir, preservar, obtener datos procesados electrónicamente a fin de dar certeza real y técnica de diversas autoridades que así lo soliciten, así como a todos aquellos titulares de dicha información”
## Frameworks 
“Marco de trabajo” que tiene los códigos informáticos hechos y para solo darle instrucciones específicas. 
# Formas de delitos cibernéticos
## Hacking 
Usar algo de forma no permitida para un beneficio 
## Cracking 
Hacer algún mal a través de alguna vulnerabilidad 
## Phishing
Pesca de inocentes 
## Evil twins
Robo de información para clonación de cuentas
## Pharming 
Páginas web falsas de estafa 
## Spamming 
Mensajes repetitivos sobre publicidad de servicios, páginas web, redes sociales, productos milagro, entre otros. 
## Ciberterrorismo
Ataques cibernpetico con fines de dañar o causar miedo a un estado o grupo. 
## Propagación de malware
Cuando un malware se distribuye en disferentes dispositivos con fines diversos como robar datos o tener control sobre los mismos.
## Pop-Up & Adware
Aparición automática de ventanas con anuncios publicitarios
## Sniffers 
Software que captura y analiza el tráfico de red que puede usarse de forma buena o mala. 
## Doxing
Investigación a fondo para recabar datos de una persona con fines buenos o malos 
## Ley Olimpia
Conjunto de reformas aplicadas en México que reconoce la violencia digital y sanciona actos que conforman delitos contra la intimidad de una persona.
## Ley Ingrid
Ley que tipifica como delito la difusión de imágenes o archivos de una carpeta de investigación para combatir la violencia de género mediática.
# Estructura de sistemas
## Monoprocesador
Utiliza una sola unidad de procesamiento 
## Multiprocesados
Utilizan varias unidades de procesamiento funcionando en simultáneo
## Distribuidos 
Estructura de múltiples equipos de computo que trabajan en conjunto.
## Embebidos
Computadoras especializadas ubicadas dentro de otros sistemas.
# Procesamiento de sistemas
## En tiempo real
Ejecutan procesos y dan una respuesta constante
## Batch 
Procesan datos en grandes volúmenes sin una interacción directa con los usuarios
## Interactivos
Permiten interacción al momento con el usuario
# Aspectos básicos de estructura de Von Neuman
## CPU
Unidad central de procesos que realiza operaciones básicas y ejecuta las instrucciones del sistema
## Unidad de control
Recupera instrucciones de la unidad de memoria para generar señales para la ejecución de instrucciones 
## Unidad de memoria
Conjunto de registros organizados de manera secuencial en donde se almacenan datos e instrucciones de programas
## Unidad de entrada y salida
Permite la obtención de información de elementos externos a la computadora 
## Hardware 
Elementos físicos de un sistema de cómputo que habla en lenguaje de impulsos eléctricos.
## Software 
Programas y sistemas operatvios que controlan al hardware y permiten la interacción con el usuario
# Tipos de memoria
## Random access memory (RAM)
Memoria volátil que almacena datos temproales mientras el sistema se encuentra operando.
### RAM Dinámica
Sobreescrita muchas veces a través de corrientes eléctricas
### RAM Estática
No requiere ser transcrita pero sigue siendo volátil y si se quita la energía se pierde la info.
## Memoria Caché
Almacena datos y comandos para el acceso constante del procesador a estos como una fomra de ahorrar recursos.
